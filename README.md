# 📊 End-to-End Data Analytics Project – Online Retail

## 🧾 Project Overview

This project is a complete **end-to-end Data Analytics solution** built using **Python, SQL, and Power BI**. It focuses on analyzing an Online Retail dataset to extract meaningful insights related to **sales performance, customer behavior, and product trends**.

The workflow covers the entire data pipeline — from **data cleaning and transformation** to **interactive dashboard visualization**.

---

## 🎯 Objectives

* Analyze overall **business performance and revenue trends**
* Identify **top-performing products and regions**
* Understand **customer behavior and retention patterns**
* Provide actionable insights for **business decision-making**

---

## 🛠️ Tech Stack

* **Python** → Data Cleaning & Preprocessing
* **SQL** → Data Storage & Querying
* **Power BI** → Data Visualization & Dashboarding

---

## 🔄 Project Workflow

### 🔹 1. Data Cleaning (Python)

* Handled missing/null values
* Removed cancelled/invalid transactions
* Created new feature:

  * **Revenue = Quantity × UnitPrice**
* Ensured data consistency and accuracy

---

### 🔹 2. Database Creation (SQL)

* Designed structured database
* Imported cleaned dataset
* Enabled efficient querying for analysis

---

### 🔹 3. Data Visualization (Power BI)

* Imported processed data into Power BI
* Created calculated measures using **DAX**
* Designed an interactive, multi-page dashboard

---

## 📊 Dashboard Structure

### 📌 1. Sales Overview

* Key KPIs (Total Revenue, Orders, etc.)
* Revenue trend over time
* Top countries by revenue
* Top-performing products

---

### 📌 2. Customer Insights

* Top customers by revenue
* Customer distribution analysis
* Repeat vs New customers
* Customer growth trends

---

### 📌 3. Product & Business Insights

* Top and low-performing products
* Product demand patterns
* Business performance indicators

---

## 🚀 Key Insights

* A small segment of products contributes to a large portion of revenue
* Certain countries dominate overall sales performance
* Repeat customers play a significant role in revenue generation
* Seasonal trends impact purchasing behavior
